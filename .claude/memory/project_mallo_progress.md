# 말로 개발 진행상황

⚠️ 이 파일은 **말로 앱 전용**입니다. 매 세션 시작 시 반드시 읽고 현재 위치 확인 후 작업할 것.

진행 표기: `[ ]`=미시작 / `[~]`=진행중 / `[x]`=완료

- [x] **1. 녹음/재생** — expo-audio로 녹음, 파일 저장, 재생. 실시간 파형 추가. 미니 실기기 확인 완료.
- [x] **2. 저장 구조 + 일정 화면** — lib/storage.ts (expo-file-system JSON)로 일정 영구 저장, 목록 화면(시각순 정렬). ※sqlite 대신 파일 JSON 채택.
- [x] **3. STT 채택** — 내장 expo-speech-recognition 한국어(ko-KR) 실시간 받아쓰기 채택. 미니에서 정확도 확인 완료. (Whisper 비교는 보류)
- [~] **4. 텍스트 구조화** — lib/parseSchedule.ts: 한국어 자연어 → {일시, 내용} 규칙 기반 파싱 구현·검증 완료. ※LLM(Claude API)은 아직, 규칙 파서로 먼저 동작.
- [ ] **5. 키 매칭 분기** — 추출된 일시로 새 일정 생성 / 기존 일정에 메모 추가 자동 분기
- [ ] **6. 메모장 화면** — 일정 안 메모 편집·부분삭제
- [ ] **7. 음성 삭제 + 미리보기** — 의도:삭제 → 삭제될 일정 보여주고 확인받기
- [ ] **8. 알람 연동** — 해당 일시에 알람/알림 등록
- [ ] **9. 예외 처리** — 날짜 인식 실패 시 안내(구현됨: 날짜·시간 없으면 저장 거부), 수동 입력 폴백/중복 처리 남음

**현재 위치** (2026-06-06): 1~3단계 완료, 4단계 규칙 파서로 동작 중. 핵심 파이프라인(음성→실시간 STT→날짜·시간 파싱→일정 저장) 미니 실기기 검증 완료.
